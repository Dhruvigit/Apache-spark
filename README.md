# Apache-spark
Pyspark practicals

The practicals cover **distributed computing**, **data processing**, **Spark SQL**, **machine learning using MLlib**, **time series analysis**, and **real-time streaming** using Apache Spark.

---

## Tools & Technologies
- Apache Spark
- PySpark
- Spark SQL
- Spark MLlib
- Spark Streaming
- Python
- Jupyter Notebook

---

## Practical List

### **Practical 1(a): Introduction to RDDs & Word Count**
- Creation of RDDs using SparkContext  
- Text processing using distributed computing  
- Word Count using `flatMap`, `map`, and `reduceByKey`

---

### **Practical 1(b): RDD Transformations – Temperature Conversion**
- RDD transformations using `map()`  
- Conversion of temperature from Celsius to Fahrenheit  
- Functional programming with RDDs

---

### **Practical 2(a): DataFrame Basics & Data Cleaning (Wine Dataset)**
- Reading CSV data into Spark DataFrames  
- Schema inference  
- Filtering and basic DataFrame operations  
- Working with real-world structured data

---

### **Practical 2(b): Feature Extraction & Transformation (Iris Dataset)**
- Creating DataFrames from in-memory data  
- Column handling and transformations  
- Basic feature exploration on Iris dataset

---

### **Practical 3: Exploratory Data Analysis (Flight Delay Dataset)**
- Loading large datasets  
- Creating temporary SQL views  
- Analyzing flight delays using Spark SQL  
- Aggregation and summary statistics

---

### **Practical 4: Spark SQL Queries & Built-in Functions**
- String, numeric, and date functions  
- Aggregate functions  
- Conditional expressions  
- Executing complex Spark SQL queries

---

### **Practical 5: Supervised Learning – Multiple Linear Regression (MLlib)**
- Feature vectorization using `VectorAssembler`  
- Training Multiple Linear Regression model  
- Model evaluation using MLlib

---

### **Practical 6: Supervised Learning – Logistic Regression (MLlib)**
- Binary classification using Logistic Regression  
- Feature preparation and labeling  
- Model training and prediction

---

### **Practical 7: Tree-Based Models – Decision Tree Classifier**
- Decision Tree model building using MLlib  
- Classification and prediction  
- Understanding tree-based learning

---

### **Practical 8: Time Series Analysis**
- Working with time-based data  
- Trend visualization  
- Forecasting future values  
- Introduction to parallel data processing concepts

---

### **Practical 9: Real-Time Data Processing with Spark Streaming**
- Spark Streaming context  
- Real-time data ingestion  
- Window operations  
- Stream processing fundamentals

---

## Learning Outcomes
- Understand distributed data processing using Spark  
- Perform structured and unstructured data analysis  
- Apply machine learning algorithms using Spark MLlib  
- Analyze time-series data  
- Process real-time data streams using Spark Streaming
